Schedulizer

Full stack application that was designed to help business owners create and schedule employee shifts and give the employees an easy way to view their schedules. Application includes full user authentication using passport, bcrypt for password encryption, node mailer for website updates, Owner/Manager/Admin accounts with different access privledges and many other features which can be viewed here: Schedulizer Github

View the deployed site at: Schedulizer

MMA News Article Scraper

Full stack MMA news article scraper that allows users to scrape articles from mmanews. Once the user clicks the scrape articles button, they are given a list of most recent news articles that can be added to their saved articles or delete articles they have already read. Each article comes with a short summary or you can read the full version with the link provided.

View the deployed site here: MMA News Scraper

React Memory Game

The goal of this game is to successfully click 12 unique cards. Each time the user selects a card the order of the cards are shuffled and the user must remember which cards they have previously picked. If a non unique card is picked the game is reset as well as your score. If the user reaches a score of 12 they win the game!

View the deployed site here: Clicky Game
